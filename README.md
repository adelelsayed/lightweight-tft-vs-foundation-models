# Lightweight TFT vs zero-shot time-series foundation models

Code and forecasts for the paper *"A Lightweight Temporal Fusion Transformer Matches
Billion-Scale Zero-Shot Foundation Models for Demand Forecasting at a Fraction of the Cost."*

> A 0.19–0.63 M-parameter, CPU-servable Temporal Fusion Transformer is statistically
> indistinguishable from four billion-scale zero-shot foundation models (Chronos, TimesFM,
> Moirai, TabPFN-TS) on two real demand datasets (NHS-EPD monthly, M5 weekly), is better
> calibrated on M5, and beats seasonal-naive where any model can — at 2–3 orders of
> magnitude lower deployment cost and with on-premises data control.

## Reproduce every number in the paper
```bash
pip install numpy pandas matplotlib pyarrow
python analysis.py        # prints all metrics + significance; writes tables/*.csv
```
`analysis.py` reads the per-seed forecasts in `results/` and regenerates the per-seed
metrics, mean±std, foundation-model metrics, the cluster-bootstrap + Diebold–Mariano
significance, and the robustness verdicts behind Tables 1–3.

## Layout
```
analysis.py                 reproduces all tables from the forecasts
paper.tex / paper.md        manuscript (LaTeX + Markdown)
figures/                    fig1 WAPE, fig2 calibration, fig3 cost-vs-accuracy
tables/                     CSVs generated by analysis.py
code/                       TFT implementation + NHS pipeline + M5 notebooks (3 seeds)
results/
  nhs/tft_forecasts/seed{42,43,44}.csv   TFT forecasts (NHS test windows)
  nhs/fm_forecasts/forecasts_*.csv       foundation-model forecasts
  m5/tft_forecasts/seed{42,43,44}.csv    TFT forecasts (3,000 shared eval windows)
  m5/fm_bench_seed42/forecasts_*.csv     foundation-model forecasts + windows_eval.csv
```

## Datasets
- **NHS-EPD** (open): https://opendata.nhsbsa.net/dataset/english-prescribing-dataset-epd-with-snomed-code
- **M5** (Kaggle): https://www.kaggle.com/competitions/m5-forecasting-accuracy

Raw M5 files are **not** redistributed here (Kaggle terms). The notebooks in `code/`
reconstruct the processed cache from the downloaded competition files.

## Foundation-model versions benchmarked
`amazon/chronos-bolt-base`, `google/timesfm-2.5-200m-pytorch`,
`Salesforce/moirai-1.1-R-large`, `tabpfn-time-series`.
